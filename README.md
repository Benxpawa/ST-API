# ST API

#### 介绍
ST API是一个采用MDUI+PHP开发的后公共API程序

#### 软件架构
HTML+MDUI+PHP


#### 安装教程

1.导入<br />
2.访问网站首页<br />
3.接口文档页面在./doc/index.php，文档在./doc/d/xxx.php<br />
4.api后端在./api<br />
